குவியத்தூரம் f கொண்ட குவிலென்ஸின் முன்பாகப் பொருளொன்று வைக்கப்பட்டுள்ளது. பெரிதாக்கப்பட்ட மெய் பிம்பம் கிடைக்க வேண்டுமெனில், குவிலென்ஸிலிருந்து பொருளை வைக்க வேண்டிய பெரும் மற்றும் சிறுமத் தொலைவுகள் யாவை? (IEE Main 2009)
